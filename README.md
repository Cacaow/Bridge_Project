Variable Information

L: total length of the bridge
n: number of divisions
dx: change in distance across beam per each iteration

x_train: location of point loads of the train
P_train: loads from each wheel of the train (point load)

E: Young's Modulus of matboard
mu: Poisson's Ratio for matboard

tf_: top flange
    tfw: top flange width
    tft: top flange thickness
bf_: bottom flange
    bfw: bottom flange width
    bft: bottom flange thickness
h_web: height of the webs
d_web:distance between webs
t_web: thickness of webs
a_web: distance between diaphragms
t_glue: thickness of glue tab
w_glue: width of glue tab

x0: position of first axle
axle_offsets: axle offsets from first axle
